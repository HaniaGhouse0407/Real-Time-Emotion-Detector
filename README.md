## 🧠 Overview

Dual-mode emotion recognition system: facial emotion detection via DeepFace/FER, and text emotion analysis via VADER + custom lexicon. Supports 7 emotion classes with per-class probability breakdown.

---

## ✨ Features

- ✅ Facial: DeepFace (VGG-Face), FER+ (ONNX), Mini-Xception models
- ✅ Text: VADER + custom sentiment lexicon
- ✅ 7 emotions: happy, sad, angry, fear, surprise, disgust, neutral
- ✅ Per-class probability bar chart
- ✅ Combined facial + text mode
- ✅ Real-time confidence scores

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/HaniaGhouse0407/Real-Time-Emotion-Detector.git
cd Real-Time-Emotion-Detector

# 2. Install dependencies
pip install -r requirements.txt

# 3. Set environment variables (if needed)
cp .env.example .env
# Edit .env with your API keys

# 4. Run
streamlit run app.py
```
